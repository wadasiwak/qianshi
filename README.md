# 解籤 ✦ 鹿港天后宮・行天宮・明治神宮 籤詩查詢

收錄三間宮廟／神社的完整籤詩，輸入籤號立即看解籤，也可以線上求籤擲筊。

- **鹿港天后宮**：天后宮一百籤，100 首（採官方線上求籤定本，含功名／婚姻／疾病等十類宮廟批註）
- **行天宮**：百首籤（關帝靈籤），100 首（逐字比對行天宮官方電子籤詩圖檔）
- **明治神宮**：大御心，30 首（明治天皇御製・昭憲皇太后御歌，不占吉凶；和歌原文＋本站中譯與解讀）

每首籤包含：籤詩原文（直式排版）、吉凶等第、典故／宮廟批註、傳統解曰、白話總解，
以及愛情婚姻／事業工作／財運／健康／考試學業／訴訟糾紛／失物行人七類分類解讀。

線上求籤流程：誠心默念 → 抽籤 → 擲筊（機率忠實模擬：聖筊 1/2、笑筊 1/4、陰筊 1/4）→ 聖筊得籤。
（明治神宮依神社慣例直接抽，不擲筊。）

其他功能：每日一籤（依日期全站同步）、籤詩全文搜尋（記得一句就能找回那首）、
單首籤分享連結（`#lukang/23`）、最近查過的籤（localStorage）。

## 開發

```
npm install
npm run dev        # http://localhost:5210
npm run content    # 從 content-raw/ 重新生成 src/content/*.ts
npm run check      # 內容完整性檢查
npm run build      # 產出 dist/
```

## 資料來源與聲明

籤詩原文與傳統解曰為民間流傳古籤（公共領域），經多來源交叉核對整理；
白話翻譯與分類解讀為本站自行撰寫，僅供參考，重大決定請以自己的判斷為主。
